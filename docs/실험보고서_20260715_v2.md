# Night calibration과 post-cutoff monitoring 실험보고서 v2

## 목적

Frozen prospective validation에서 가장 낮았던 night segment F1 0.7960이 threshold 조정으로 개선 가능한지 leakage 없이 검증하고, cutoff 이후 snapshot을 기존 cohort에 섞지 않은 채 input drift를 감시한다.

## 데이터와 분할

- Frozen cutoff: `2026-07-13T14:15:03+09:00`
- Frozen label panel: 817,668행
- 기존 final split: train 75%, test 25%
- Threshold 학습용 train 내부 split: fit 489,636행, calibration 123,012행
- Final holdout: 205,020행
- 시간 경계: fit 종료 `2026-07-07T21:15:03+09:00`, calibration `2026-07-07T22:15:03+09:00`~`2026-07-10T00:15:03+09:00`, test 시작 `2026-07-10T01:15:03+09:00`
- Post-cutoff monitoring: 21 snapshots, 50,652행

## 방법과 파라미터

- Model: class-balanced logistic inventory model
- Feature: 기존 full 9 features
- Threshold grid: 0.05~0.95, step 0.025
- Global/night threshold는 calibration split의 F1, precision, 0.5 근접도 순으로 선택
- Candidate 채택 gate:
  - calibration과 final test에서 전체 F1 하락 0 이하
  - calibration과 final test에서 night F1 개선 0.001 이상
- Final test는 threshold 재튜닝에 사용하지 않고 non-degradation acceptance에만 사용
- Post-cutoff drift threshold: shortage-rate diff 0.05, PSI 0.25, hour TV 0.20, station coverage 0.95

## Threshold calibration 결과

선택된 logistic threshold는 global 0.900, night 0.675다.

| calibration | persistence | night | 0.8121 | 0.8330 | 0.8224 | 0.7068 | 0.0655 |

Calibration night F1 개선은 약 0.0009로 최소 기준 0.001에 미달했다. Final holdout에서도 candidate 전체 F1은 0.0012, night F1은 0.0007 낮았다. Logistic AP는 높지만 binary action F1과 Brier가 persistence보다 나빠 서로 다른 사용 목적을 합치지 않는다.

## Class balance와 시간대 원인

Final test의 night shortage rate는 0.1581로 non-night 0.1320보다 높다. Positive-to-negative ratio도 night 0.1877, non-night 0.1521로 차이가 있다.

Current shortage state와 next-snapshot target이 바뀌는 transition rate는 night 6.40%, non-night 4.18%다. Persistence가 night에서 약해지는 주된 관측 근거는 threshold 부족만이 아니라 상태 전이 빈도가 더 높다는 점이다.

## Post-cutoff drift 결과

| Metric | 값 | 기준 | 상태 |
|---|---:|---:|---|
| Shortage-rate absolute difference | 0.0122 | ≤0.05 | PASS |
| Inventory-pressure PSI | 0.0017 | ≤0.25 | PASS |
| Hour-distribution TV | 0.1765 | ≤0.20 | PASS |
| Station coverage | 1.0000 | ≥0.95 | PASS |

Frozen reference 340 snapshots과 monitoring 21 snapshots을 분리한 상태에서 4/4가 통과했다. Hour TV는 기준에 상대적으로 가까우므로 snapshot이 24시간 이상 누적된 뒤에도 계속 관찰한다. 어떤 check도 model이나 threshold를 자동 변경하지 않는다.

## 판단

- Decision: `KEEP_PERSISTENCE_BASELINE`
- Binary action threshold는 persistence 0.5를 유지한다.
- Logistic probability는 높은 AP를 활용하는 ranking 보조 근거로만 유지한다.
- Night 문제는 event, service 상태, station transition feature를 확보하기 전까지 자동 action 확대 근거로 사용하지 않는다.
- 모든 결과는 predictive validation과 input monitoring이며 현장 재배치의 인과효과가 아니다.
