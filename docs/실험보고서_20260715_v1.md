# Station prospective quality hardening 실험보고서 v1

## 목적

단일 25% temporal holdout과 정적 self-score만으로 portfolio quality를 판단하던 한계를 줄인다. Frozen Citi Bike station cohort에서 시간 경과 안정성, feature 기여, 분포 drift, 운영 segment 실패를 재평가하고, 검증 산출물과 최신 테스트가 모두 있을 때만 quality floor를 올린다.

## 데이터와 방법

- 데이터: `station_shortage_label_panel.csv`의 label 817,668행, 2,412 stations, 339 label timestamps
- Frozen cutoff: `2026-07-13T14:15:03+09:00`
- Source/frozen/excluded snapshots: 361 / 340 / 21
- Target: `bike_shortage_next_snapshot`
- Final split: timestamp 기준 train 75%, test 25%
- Rolling-origin: 초기 50% expanding train, 3 folds
- 모델: persistence baseline, station-hour profile, class-balanced logistic inventory model
- Logistic 설정: `max_iter=500`, `class_weight=balanced`, `random_state=42`, `StandardScaler`
- Ablation: full 9 features, current-state flag 제거 7 features, temporal-only 3 features
- Drift threshold: shortage-rate difference 0.05, inventory PSI 0.25, hour TV 0.20, station coverage 0.95
- Failure segments: all, commute peak, non-commute, weekend, night, high inventory pressure

## 결과

### Final holdout

| 모델 | F1 | Average precision | Brier |
|---|---:|---:|---:|
| persistence baseline | 0.8286 | 0.7102 | 0.0478 |
| logistic inventory | 0.6001 | 0.8772 | 0.0537 |
| station-hour profile | 0.4529 | 0.4803 | 0.0994 |

Persistence는 binary action F1과 calibration에서 우수했고, logistic은 ranking metric인 average precision에서 우수했다. 하나의 metric으로 두 사용 목적을 합치지 않는다.

### Rolling-origin

- 3 folds, 9 model-fold rows를 모두 평가했다.
- Fold별 최고 F1은 0.8830, 0.8362, 0.8238이다.
- Fold-best F1 평균은 0.8477, 최저는 0.8238이다.
- 각 fold에서 persistence baseline이 F1 기준 최상위였다.

### Feature ablation

| Feature set | F1 | Average precision |
|---|---:|---:|
| full | 0.6001 | 0.8772 |
| no current-state flags | 0.6087 | 0.8748 |
| temporal-only | 0.2442 | 0.1481 |

Current-state flag 제거 영향은 작지만 temporal-only는 크게 하락했다. Shortage ranking의 핵심은 시간 feature보다 capacity와 현재 inventory 수준이다.

### Drift와 failure audit

- Shortage-rate absolute difference: 0.0115 ≤ 0.05
- Inventory-pressure PSI: 0.0021 ≤ 0.25
- Hour-distribution TV: 0.0960 ≤ 0.20
- Station coverage: 1.000 ≥ 0.95
- Drift gate: 4/4 PASS
- Segment 최저 F1: night 0.7960

## Quality 판단

기존 10개 category score는 정적 숫자만 바꾸지 않았다. Prospective `PASS`, frozen cohort, rolling 3 folds, ablation 3 rows, drift 4/4, failure segment 5개 이상, public evidence `GO`, README/figure/final-report contract, 최신 source보다 새로운 JUnit 55 tests PASS가 모두 확인될 때만 각 category를 96.0으로 승격한다. 근거가 빠지면 기존 92~95 score로 돌아간다.

- Active floor 95.8 기준 strict Sunday validator: PASS, minimum 96.0 > 95.8
- Quality floor ratchet: 95.8 → 96.0
- Ratchet 이후 floor-mode validator 기준: minimum 96.0 ≥ 96.0

## 결론과 다음 단계

Quality hardening은 통과했다. Persistence baseline을 binary action 기준으로 유지하고 logistic probability는 ranking 보조 근거로만 사용한다. Night segment는 전체보다 약한 F1 0.7960이므로 자동 action threshold를 확대하지 않고, post-cutoff monitoring drift와 함께 별도 calibration 대상으로 둔다. 모든 값은 predictive validation이며 실제 재배치의 인과효과나 현장 성과를 뜻하지 않는다.
