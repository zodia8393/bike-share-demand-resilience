# 문서 언어 기준

이 저장소의 본문 문서는 한글로 작성합니다. 코드 식별자, 파일 경로, 모델명,
지표명, 단위는 원문 표기를 유지합니다.

## 적용 파일

- `README.md`
- `docs/` 하위 문서
- `reports/final_report.md`
- `reports/model_card.md`
- `reports/data_source_and_contract.md`

## 기록 방식

- 실행한 작업과 결과를 먼저 적습니다.
- 수치가 있는 내용은 파일명이나 지표명을 함께 남깁니다.
- 공개 데이터의 한계와 운영 적용 전제는 별도 문단으로 분리합니다.
- 템플릿 문장은 실제 실행 결과에 맞지 않으면 남기지 않습니다.
